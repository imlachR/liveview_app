Diferença Dados os conjuntos A = {1, 2, 3, 4}, B = {4, 5} e C = {3, 4, 5, 6, 7}, determine o número de subconjuntos de (A - B) ∩ C. Dados os conjuntos A = {1, 2, 3, 4, 5}, B = {1, 2, 4, 6, 8} e C = {2, 4, 5, 7}, obtenha o conjunto X tal que X ⊂ A e A - X = B ∩ C.

Dados os conjuntos A e B, podemos determinar um conjunto cujos elementos pertencem ao conjunto A e não pertencem ao conjunto B. Esse conjunto é chamado diferença entre A e B e indicado por A - B, que se lê “A menos B”. Assim, define-se:

<%# sav__text %>

A - B = {x | x ∈ A e x ∉ B}

<%# sav__media-grid %>
  • A ⊂ B
  • A e B disjuntos
  • B ⊂ A

Dados os conjuntos A = {1, 2, 3, 4, 5}, B = {3, 4, 5, 6}, C = {2, 3} e D = {0, 7, 8}, temos:


  • A - B = {1, 2}
  • B - A = {6}
  • C - D = {2, 3}, pois, como C ∩ D = ∅,C - D = C.
  • C - A = ∅, pois C ⊂ A.
  • D - D = ∅
